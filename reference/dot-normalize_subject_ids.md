# Normalize Subject IDs

Vectorized wrapper for
[`.normalize_subject_id()`](https://bbuchsbaum.github.io/openneuroR/reference/dot-normalize_subject_id.md).

## Usage

``` r
.normalize_subject_ids(ids)
```

## Arguments

- ids:

  Character vector of subject IDs.

## Value

Character vector with all IDs prefixed with "sub-".
