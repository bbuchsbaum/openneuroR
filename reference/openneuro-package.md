# openneuro: Programmatic Access to OpenNeuro Datasets

Search, explore, and download datasets from 'OpenNeuro'
<https://openneuro.org>, the largest open neuroimaging data repository.
Queries the 'OpenNeuro' GraphQL API to discover datasets by modality,
diagnosis, or keyword; inspect snapshots, files, and subject lists; and
download full datasets or selected subsets via HTTPS, Amazon S3, or
'DataLad'. Downloaded data are cached locally so subsequent requests
skip already-fetched files.
