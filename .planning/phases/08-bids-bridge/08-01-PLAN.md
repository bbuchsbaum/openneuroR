Create R/bids.R with:

1. **on_bids(handle, fmriprep = FALSE, prep_dir = "derivatives/fmriprep")** - Main function:
   - Use `rlang::check_installed("bidser", reason = "to create BIDS project objects from OpenNeuro datasets")` FIRST
   - Validate input is `openneuro_handle` class with informative error (suggest `on_handle()`)
   - Auto-fetch if handle$state == "pending" - capture return: `handle <- on_fetch(handle)`
   - Get path via `on_path(handle)`
   - Call `.validate_bids_structure(path)` to check dataset_description.json exists
   - Handle derivatives: if both fmriprep=TRUE and custom prep_dir, prep_dir wins
   - Check derivatives path exists with `.check_derivatives_path()`, warn if missing
   - Brief cli message: `cli::cli_alert_info("Creating BIDS project from {.val {handle$dataset_id}}")`
   - Call `bidser::bids_project(path = path, fmriprep = use_fmriprep, prep_dir = prep_dir)`
   - Return the bids_project directly (not wrapped)

2. **.validate_bids_structure(path)** - Internal helper:
   - Check `fs::path(path, "dataset_description.json")` exists
   - If missing, `cli::cli_abort()` with class "openneuro_bids_error"
   - Include helpful message about BIDS requirements

3. **.check_derivatives_path(path, fmriprep, prep_dir)** - Internal helper:
   - If not requesting derivatives (fmriprep=FALSE and default prep_dir), return NULL
   - Build deriv_path = fs::path(path, prep_dir)
   - If path doesn't exist, `cli::cli_warn()` and return FALSE
   - If exists, return TRUE

4. **Roxygen documentation** for on_bids():

   - @return A bids_project object from bidser
