   - If !interactive(): suppress progress bars, only print summary at end
   - If interactive() and !quiet: show progress bar

   Progress bar format (using cli::cli_progress_bar):
   ```r
   cli::cli_progress_bar(
     name = "Downloading",
     total = nrow(files_df),
     format = "{cli::pb_bar} {cli::pb_current}/{cli::pb_total} files [{.download_rate}]"
   )
   ```

   For each file:
   - Build dest_path: `fs::path(dest_dir, file$full_path)`
   - Check if skip (exists with correct size, force=FALSE): increment skipped counter
   - Otherwise: call `.download_single_file()` via `.download_atomic()`
   - Update progress: `cli::cli_progress_update()`
   - Track: downloaded_count, skipped_count, failed_files, total_bytes

   On completion: call `.print_completion_summary()`

   Return list(downloaded = n, skipped = n, failed = list(), total_bytes = n, dest_dir = path)

2. `.format_bytes(bytes)` - Human-readable byte formatting
   - < 1024: "{n} B"
   - < 1024^2: "{n/1024} KB"
   - < 1024^3: "{n/1024^2} MB"
   - else: "{n/1024^3} GB"
   - Round to 1 decimal place

3. `.print_completion_summary(result, quiet)` - Print completion message
   - If quiet=TRUE: return invisible
   - Format: "Downloaded {n} files ({size}) to {dest_dir}"
   - If skipped > 0: add "Skipped {n} existing files"
   - If any failed: add warning "Failed to download {n} files: {names}"
   - Use cli::cli_alert_success for main message
   - Use cli::cli_alert_info for skipped
   - Use cli::cli_alert_warning for failures

#' @param tag Snapshot version tag. If NULL (default), uses latest snapshot.

#'   (default), downloads all files.
#' @param dest_dir Destination directory. If NULL (default), creates
#'   `./dataset_id/` in the current working directory.
#' @param quiet If TRUE, suppress all progress output. Default FALSE.
#' @param verbose If TRUE, show per-file progress in addition to overall
#'   progress. Default FALSE.
#' @param force If TRUE, re-download files even if they exist with correct
#'   size. Default FALSE.
#' @param client An openneuro_client object. If NULL, creates default client.
#'
#' @return Invisibly returns a list with:
#'   \describe{
#'     \item{downloaded}{Number of files downloaded}
#'     \item{skipped}{Number of files skipped (already existed)}
#'     \item{failed}{Character vector of failed file names}
#'     \item{total_bytes}{Total bytes downloaded}
#'     \item{dest_dir}{Path to destination directory}
#'   }

#' @export
#' @examples

#' # Download full dataset

#' on_download("ds000001", files = "sub-0[1-3].*")

#'
#' # Force re-download of existing files
