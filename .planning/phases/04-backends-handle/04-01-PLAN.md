  <action>
Create R/backend-detect.R with backend availability detection:

1. `.backend_available(backend)` - Check if specific backend is available:
   - "s3": `nzchar(Sys.which("aws"))`
   - "datalad": `nzchar(Sys.which("datalad")) && nzchar(Sys.which("git-annex"))`
   - "https": Always TRUE
   - Return FALSE for unknown backends

2. `.backend_status` - Session-cached detection using local environment:
   ```r
   .backend_status <- local({
     cache <- list()
     function(backend, refresh = FALSE) {
       if (refresh || is.null(cache[[backend]])) {
         cache[[backend]] <<- .backend_available(backend)
       }
       cache[[backend]]
     }
   })
   ```

3. `.find_aws_cli()` - Find AWS CLI in PATH or common locations:
   - Check Sys.which("aws") first
   - Then check common paths: /usr/local/bin/aws, /opt/homebrew/bin/aws, ~/.local/bin/aws
   - On Windows: C:/Program Files/Amazon/AWSCLIV2/aws.exe
   - Return path or empty string

4. Add processx to DESCRIPTION Imports (needed for CLI execution)

All functions are internal (not exported). Use roxygen2 @keywords internal.
